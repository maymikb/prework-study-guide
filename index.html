Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • ul means the unordered list represented by bullet points, there is not limit to the length of the list
  • Each individual item under ul tag must be enclosed within an li tag.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Using, * applies a rule to all the elements on the page(to v the same style properties), as opposed to repeating
  • a comma can be used to separate 2 selectors in one CSS rule in order to affect both and avoid repetition

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • EVERY git command must be preceeded by the word "git"
  • Git is a version control system making it easier to find bugs, and to collaborate with others
  • It is best practice to sync your version of the code that you are working on with github
